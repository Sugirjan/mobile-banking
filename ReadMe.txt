Gender - User's gender
Age  - User's age
City - User's city
Education - User's education level
Occupation - User's occupation
Monthly_Income - Monthly Income of user
IM_Usage_period - Mobile with internet usage period
Experience - Level of mobile experience
Bank - Bank that use for Mobile Banking App
Source - Source to know the Mobile Banking App
Bank_Account_Type - Type of the bank account
Usage_Time - How often make transaction
Distance - Distance between Bank branch and living place
Service - Service that use in Mobile App
TimeSaving - Perception on saving time using Mobile App
Easyness - Perception on easyness using Mobile App to carry out banking operations
UI_Design - Perception on Mobile App usage friendly
Network_Connectivity - Trust for Network Connectivity while doing a transaction using mobile banking
Cost_effective - Perception on cost-effective mobile banking as compare to visiting banks
Availabilty - Perception on using mobile banking because of its 24 X 7 availability
Security - Satisfaction with the level of data and information security provided by the bank 
Comfortable - Perception on comfortable using new technologies like mobile banking
Awareness -  Awareness of all the mobile banking app services
Usage - Usage of services in mobile App
Aware_Charges - Aware of the charges from banks to use the mobile banking app
Charges - Feeling about charges
Satifaction - Overall satisfation
Recommendation - Have recommended mobile banking to others
